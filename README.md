# Nova Orders Dashboard

Мини-дашборд заказов для тестового задания AI Tools Specialist.

## Стек
- RetailCRM — хранение заказов
- Supabase — база данных
- Vercel — хостинг дашборда
- Telegram Bot — уведомления о крупных заказах

## Что сделано
- Загружены 50 тестовых заказов в RetailCRM через API
- Скрипт синхронизации RetailCRM → Supabase
- Дашборд с графиком заказов по городам: https://nova-dashboard-ivory.vercel.app/
- Telegram-бот уведомляет о заказах больше 50 000 ₸

## Где застряла и как решила
- **orderType не найден** — убрала поле из запроса к API
- **customFields возвращался как список** — добавила проверку isinstance
- **сумма заказа = 0** — оказалось нужно добавлять товары в заказ в RetailCRM, не просто сумму
- **Git не установлен** — загрузила файлы напрямую через GitHub
